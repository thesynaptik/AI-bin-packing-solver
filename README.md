# AI Bin Packing Solver

## Overview

This project solves the Bin Packing Problem using two different optimization approaches:

- Backtracking Algorithm (Exact Solution)
- Genetic Algorithm (Heuristic AI Solution)

The application includes a graphical user interface built with Tkinter that allows users to compare algorithm performance, visualize bin allocation, and analyze execution time.

## Features

- Exact optimization using Backtracking
- AI-based optimization using Genetic Algorithms
- Interactive GUI using Tkinter
- Execution time comparison
- Bin visualization
- Performance analysis

## Technologies

- Python
- Artificial Intelligence
- Genetic Algorithms
- Backtracking
- Optimization Algorithms
- Tkinter
- Matplotlib

## Algorithms

### Backtracking
Searches for the optimal arrangement while minimizing the number of bins through recursive exploration and pruning.

### Genetic Algorithm
Uses:
- Population Generation
- Fitness Function
- Selection
- Crossover
- Mutation

to find near-optimal solutions efficiently.
